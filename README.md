ControlPointsSelector (QGIS Plugin)

Плагин для QGIS 3.x, который помогает выбирать контрольные точки отбора проб в зависимости от направления ветра и текущей погоды.
Интерфейс реализован в виде док-панели, основанной на шаблоне QGIS Plugin Builder.

Основные возможности:
Получение текущих погодных данных (температура, скорость и направление ветра) по координатам выбранной площадки — через OpenWeatherMap API.
Автоматический отбор подходящих точек контроля по направлению ветра.
Подсветка подходящих точек на карте QGIS.
Отображение и изменение счётчика выездов ridesCounter (+ / - и ручной ввод).
Экспорт подходящих точек в Excel (.xlsx) с заголовками и форматированием.
